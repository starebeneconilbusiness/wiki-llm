# Paperclip – AI Agent Orchestration for Work

**Paperclip** is an open-source, self-hosted app for managing AI agents as a structured team — like running a company, not just prompting tools.

## Quickstart
- Open source (MIT-licensed), self-hosted, no account required.
- Install via: `npx paperclipai onboard --yes`

## Core Concept
"The mental model is a company you are running, not a tool you are using." — @yashns1

## Key Features

### 1. Org Chart
- Hierarchical structure: CEO, CTO, engineers, marketers, etc.
- Any agent (Claude, Codex, Cursor, OpenClaw, etc.) can be hired.
- Roles, reporting lines, and job descriptions included.

### 2. Goal Alignment
- Every task traces back to a mission.
- Agents understand what to do and why.

### 3. Cost Control
- Monthly budgets per agent (e.g., CEO: $60, CMO: $40).
- Auto-pause at 100% budget; soft warning at 80%.

### 4. Ticket System
- All work happens via structured tickets with owners, status, and threads.
- Immutable audit log — append-only, no edits or deletions.

### 5. Governance
- You approve all hires and strategies.
- Pause, resume, override, reassign, or terminate any agent at any time.

### 6. Heartbeats
- Agents wake on schedule (e.g., every 4h, 8h, 12h).
- Delegation flows up/down org chart automatically.

### 7. Model-Agnostic
- Works with any agent runtime: Claude, Codex, Gemini, Cursor, OpenClaw, Hermes, Pi, OpenCode, etc.
